# プログラミング2：演算と型・条件分岐に関する課題

2026年度 プログラミング2の講義における、演算、データ型、および条件分岐に関する演習プログラムです。

## 課題内容

### 演算と型に関する問題
1. **3つの実数の平均（実数表示）**: キーボードから入力された3つの実数の平均値を実数（`double`型）で出力します。
2. **3つの実数の平均（整数表示）**: 上記平均値を整数（`int`型）にキャストして出力します。
3. **合計金額の計算**: リンゴ（100円）、オレンジ（88円）、苺（398円）の各購入数から、税込の合計金額を出力します。
4. **チーム分けと余り人数の計算**: 全体人数から野球（9人）とバレー（6人）のチームを作った際の、参加できない人数を出力します。
5. **三角形の面積計算**: 入力された実数の底辺と高さから、三角形の面積を出力します。

## 実行環境

- コンパイラ: gcc (または同等のCコンパイラ)
